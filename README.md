# Mission 2 - Project L5

Welcome to Mission 2! This project is part of the Level 5 advanced track.

## 📋 Overview

This mission focuses on building a comprehensive **Full-Stack Payment Application** with routing, testing, and utility components. The project demonstrates advanced backend development, API design, and test-driven development practices.

## 📁 Project Structure

```
mission-2/
├── components/
│   ├── routes/              # API route handlers
│   ├── testing/             # Test suites
│   ├── utilities/           # Helper functions
├── package.json             # Project dependencies
├── server.js                # Express server
├── MISSION.md               # Mission requirements
├── README.md                # Project documentation
└── AGENTS.md                # Agent documentation
```

## 🚀 Quick Start

### Prerequisites
- Node.js 16+
- npm or yarn

### Installation

```bash
npm install
```

### Running the Server

```bash
npm start
```

The server will start on the configured port (default: 3000)

### Development Mode

```bash
npm run dev
```

### Running Tests

```bash
npm test
```

All tests in the `components/testing/` directory will be executed.
